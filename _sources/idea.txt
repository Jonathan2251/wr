iCareHome
~~~~~~~~~

  - I Care Home System, Internet Care Home System
  
  - A system which watch old men or babies.
  
  - System scenario:
  
    - A box (host) connect to ethernet socket. Person wear our device (wear device). Host contact device every x minutes. 
	  If host lose device in satisfy the condition of set, then send message to his family or our system server.
	  Our system server will phone the customer's family if the customer want it.
	  
	- The condition can be configured by our engineer and save in system server. Or user change condition on system server. Host keep cache the condition too. For example, the condition can be:
	
	  - Lost connect in one day (lose connection of time). 
	  
	- Home can install camera detecter to detect the person instead of using wear device.
	  
  - Channel for sale and service:
  
    - Find salesman in each area and they get money only from the selling each product and the system work well in 3 months. The salesman need to install our product at customer's home.
	
	- The employee of electronic devices store can be our salesman too.
	  
	  
iMedical
~~~~~~~~

A medical system in cloud which offers user to find the problems about his body. This is a non-profit system.

- System scenario:

  - User can input their symptoms and find the possible problems about his body. Through interative question and answer from system to user, the rate of possible problems shown to user for reference.
  
  - User can decide to save to not save their information on system any time. Or delete his information totally any time.
  
  - If user allow his information to be used in medical research, then he can share the revenue we get from medical research.
  
  - The revenue can come from medicine research, or doctor research, or the system is used to help doctor doing their work.
  
  - No any ID for personal identication saved in system.
  
  - Doctor connect with our system can help our system smarter, and doctor can get more correct judgement in his work.
